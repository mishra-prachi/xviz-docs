Bullet Chart

The Bullet Chart for Power BI is a variation of a bar chart developed as a replacement for gauges and meters. The Bullet chart has the following advantages over the gauges. It saves space and requires less real estate, as it can be oriented horizontally and vertically based on the space available. The bullet chart displays multiple measures in a single visual. The custom visual is designed in such a way that the information present in the chart is conveyed easily.

Conditional Formatting#

The conditional formatting provides robust alerting capabilities and convey the state of the KPI whether it is doing good or bad.

Reverse Axis and Legends RTL support#

Reverse Axis is available for both X and Y axis and RTL Support enables readability of languages such as Arabic

Marker Customization#

The markers in the custom visual can display 2 benchmark values and the style of the marker can be customized as per enterprise need.

Comparison Band Customization#

The custom visual provides the ability to have another series option to compare with the actual value both on a comparative scale and a qualitative scale

Advanced Analytical Features#

This feature sets a common benchmark using the horizontal or vertical reference line under the analytics section

Summary Table#

Instead of Complex workarounds, Summary Table feature can be toggled to view the chart data in a tabular format

Orientation#

The Bullet Chart for Power BI enables you to choose between horizontal or vertical orientation

Detailed tooltip#

The tooltip option displays a consolidated view to display all the data at once and we can also toggle comparison band tooltip

Use Cases#

Horizontal and Vertical orientation of the visual helps it to display multiple measures at the same time.

Performance of a product can be compared with one or more alternative measures

Multiple Qualitative Bands gives an overview of the target achieved as Good, Average or Bad

Targets can be identified by applying conditional formatting to the bands
